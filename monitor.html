Monitors

Tweakpane also has a feature for monitoring value changes. Use addMonitor() to add components.

Multiline

multiline option provides multiline log components.

const pane = new Tweakpane();
pane.addMonitor(PARAMS, 'wave', {
  multiline: true,
});

Count

You can use count option to change a buffer size.

const pane = new Tweakpane();
pane.addMonitor(PARAMS, 'wave', {
  count: 10,
});

Interval

To change an interval of monitors, set interval option in milliseconds. The default value is 200(ms).

const pane = new Tweakpane();
pane.addMonitor(PARAMS, 'time', {
  interval: 1000,
});

Graph

{view: 'graph'} option for number parameters provides a graph component. Pass min and max for value range.

const pane = new Tweakpane();
pane.addMonitor(PARAMS, 'wave', {
  view: 'graph',
  min: -1,
  max: +1,
});
