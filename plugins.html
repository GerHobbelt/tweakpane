Plugins

Expand Tweakpane world.

Interval

Range slider control for interval object value {min, max}.

const PARAMS = {
  size1: {min: 16, max: 48},
  size2: {min: 16, max: 48},
};

// Text inputs by default
pane.addInput(PARAMS, 'size1');

// Additional range slider with `min` and `max` option
pane.addInput(PARAMS, 'size2', {
  min: 0,
  max: 100,
  step: 1,
});

tweakpane-plugin-interval by cocopon

Camerakit

Enjoyable camera flavored controls.

const PARAMS = {
  flen: 55,
  fnum: 1.8,
  iso: 100,
};

// ...

pane.addInput(PARAMS, 'fnum', {
  // Use a ring view of Camerakit
  plugin: 'camerakit',
  view: 'ring',
  // Appearance of the ring
  series: 1,
  // Configuration of the scale unit
  unit: {ticks: 10, pixels: 40, value: 0.2},
  // Hide a text input and widen the ring
  wide: true,
  // Number constraints
  min: 1.4,
  step: 0.02,
});

// ...

Create your own plugin

From version 2.0.0, you can provide custom input/monitor components for various user inputs with the plugin system.

Tweakpane.registerPlugin({
  type: 'input',
  plugin: {
    ...
  },
});
